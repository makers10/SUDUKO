Welcome to Sudoku

Sudoku is a logic-based number placement puzzle. The objective is to fill a 9×9 grid with digits so that each column, each row, and each of the nine 3×3 sub-boxes contain all of the digits from 1 to 9.

123456789
456789123
789123456

Each row must contain 1–9 without repetition

📏

The Rules

1
Rows: Each row must contain the numbers 1–9 with no duplicates.
2
Columns: Each column must contain the numbers 1–9 with no duplicates.
3
3×3 Boxes: Each of the nine 3×3 sub-grids must contain the numbers 1–9 with no duplicates.
4
Given Numbers: Some cells are pre-filled. These cannot be changed.
💡

Tips & Strategies

🔍

Scanning

Look for rows, columns, or boxes that are nearly complete. The missing numbers are easier to figure out.

✏️

Pencil Marks

Use the Notes mode to write small candidate numbers in cells when you're unsure.

🎯

Elimination

If a number appears in a row and column, it can't appear again. Use this to eliminate possibilities.

🧠

Single Candidate

If only one number can fit in a cell after elimination, that must be the answer!

🎮
